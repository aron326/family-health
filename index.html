🔧 設定 GitHub Gist 雲端儲存

完成設定後，健康資料將自動同步至您的私人 GitHub Gist，跨裝置共用。

登入 GitHub，前往 Settings → Developer settings → Personal access tokens → Tokens (classic)，點 Generate new token
勾選 gist 權限，其他不需要勾選，產生後複製 Token
將 Token 貼入下方欄位，點「連線並初始化」，系統會自動建立私人 Gist
→ 直接前往建立 Token 頁面 → 查看我的 Gists
資料儲存於您的私人 GitHub Gist，Anthropic 不儲存任何健康資訊。以上建議僅供參考，請諮詢專業醫療人員。
